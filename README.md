# No Spend Tracker

Mini app pessoal para acompanhar dias sem gastar dinheiro.

## Páginas
- `index.html` → contador principal
- `history.html` → histórico de gastos (resets)

## Dados
Os dados são armazenados localmente no navegador (`localStorage`).

## Deploy
Compatível com:
- GitHub Pages
- Netlify
- xTiles (iframe)
